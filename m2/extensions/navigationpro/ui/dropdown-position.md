# Dropdown Position

> This setting is available for the **First Level Dropdowns** only in
> [Menu Settings](/m2/extensions/navigationpro/backend/menu-settings/#dropdown-settings)
> form

Following positions are available:

 -  Stick to Left
 -  Center
 -  Stick to Right
