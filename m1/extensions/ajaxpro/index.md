# AjaxPro

![Ajax popup while bundle product adding](/images/m1/extensions/ajax-pro/frontend.png)

Help your customers add the product to the cart and edit shopping cart content faster without page reload. Our module comes with ajax functionality such as: auto-loading products on category page, add-to-wishlist and add-to-compare blocks, the display of different dialog window types while adding the item to the cart, setup of Login form block and Cart block in the header store block, ability to specify effect settings for Ajax blocks and for message window in ajax processing block.
